# kaomoji

> facial expressions using text characters

Organized by emotion. Search tags include ASCII descriptors and approximations.

# ASCII-fication for searchability

## approximants (prefix to avoid trigger overlap)

- 'w    - ω ꒳ ⩊
- 'A    - ∀
- '>    - ≧ ᗒ ＞ ゝ
- '-    - ￣ 눈 ╥ 〒 ¯
- '3    - ε Σ
- '^    - ＾ ヘ ︿ ‸ 人 八
- '~    - ～ ﹏
- '#    - メ ╬ ҂ ‡ ＃
- 'x    - メ × ﾒ ༝
- '+    - ＋ ±
- 'v    - ˇ
- '...  - ・・・ 。。。 ｡｡｡
- 'I    - ｪ ｴ ェ
- 'u    - ∪ μ
- '*    - ☆ *
- 'D    - Д
