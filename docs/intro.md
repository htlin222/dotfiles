# 書名構想

- 中文《終端人生：純 CLI 開發者的完全指南》
- 英文《The Terminal Way》 （用於 slug, ...）

---

## 架構規劃

### 第一部：哲學與基礎

1. **為什麼選擇終端？** — 效率、可重複性、遠端工作、專注力
2. **Unix 哲學與工具思維** — 小工具組合、文字流、管道
3. **打造你的 dotfiles** — 版本控制你的環境、跨機器同步

### 第二部：核心工具

4. **Shell 的選擇與調教** — Bash/Zsh/Fish 的取捨與設定
5. **終端模擬器** — Alacritty、Kitty、WezTerm 的特性比較
6. **多工管理：tmux 深入指南** — session、window、pane、腳本化
7. **編輯器之道：Neovim** — 從入門到 LSP 全配置
8. **版本控制的日常** — Git CLI 與 lazygit

### 第三部：效率工具鏈

9. **現代 CLI 工具集** — fzf、ripgrep、fd、eza、zoxide
10. **模糊搜尋的魔法** — fzf 整合到各種工作流
11. **文字處理三劍客** — awk、sed、grep 實戰

### 第四部：語言與專案

12. **多語言版本管理** — asdf/mise 的運用
13. **容器化開發** — Docker CLI 工作流
14. **遠端開發** — SSH、Mosh、遠端 Neovim

### 第五部：進階整合

15. **打造個人 CLI 工具** — Shell script 與小型工具
16. **自動化與腳本** — 讓重複的事消失
17. **除錯與效能分析** — 終端下的 profiling 工具

### 附錄

- 推薦 dotfiles 範例
- 快捷鍵速查表
- 從 VS Code 遷移指南

- **QR Code 連結影片**，看實際操作
